## 一、評判

### 評分架構

**台上發言（每位辯員 100 分）**

| 項目 | 分數 |
| :--- | :--- |
| 內容 | 40 |
| 辭鋒 | 30 |
| 組織 | 20 |
| 風度 | 10 |

- 另設**全隊內容連貫 5 分**（評估各辯位之間的內容一致性）

**自由辯論（全隊 55 分）**

| 項目 | 分數 |
| :--- | :--- |
| 內容 | 20 |
| 辭鋒 | 15 |
| 組織 | 10 |
| 風度 | 5 |
| 合作 | 5 |

**隊伍總分**：4 位辯員台上發言 + 自由辯論 + 內容連貫 − 扣分，滿分 **460 分**。

### 最佳辯論員
從兩隊所有台上辯員中選出：
1. 比較各評判給予每位辯員的名次，**名次總和最低者**勝（名次越低越好）。
2. 若名次總和相同，比較**平均得分**，較高者勝。
3. 若平均得分仍相同，由評判團商議決定。

### 勝負判定
- **一位評判**：總分高者勝。
- **多位評判**：票數多者勝。
- **平票**：經所有評判及主席同意，可增設自由辯論環節重新評分（設 2 分鐘準備時間）。

### 需要執行的扣分（由主席通知後執行）

| 情況 | 扣分 |
| :--- | :--- |
| 辯員逾時（從時限後第 15.01 秒起，每 5 秒扣 3 分，上限 15 分） | 3 分／5 秒 |
| 使用外語（非書名、人名、專有名詞） | 每次酌量 1–5 分 |
| 名單變動（臨時頂替或兼任非報名辯位） | 每個辯位 5 分 |
| 其他違規（由主席裁定） | 酌量 1–8 分 |

## 二、賽會人員

### 賽事主席職責
- **賽前**：向在場人士解釋賽規及宣讀辯題；提醒辯員比賽前提出疑問，比賽開始後不受理。
- **賽中**：
  - 邀請辯員發言，維持賽事流程。
  - 發現以下情況須即時處理並通知評判：
    - 辯員使用外語（非書名、人名、專有名詞）
    - 辯員作人身攻擊、使用粗言穢語或具冒犯性行為
    - 辯員攜帶違禁設備（具計時、通訊、錄音/攝影功能）
    - 台上辯員與台下觀眾溝通（直接取消參賽資格）
  - 辯員發言時間完畢後第 **40.01 秒**，須勒令其立即停止發言。

### 計時員職責

| 時間點 | 鳴鐘次數 |
| :--- | :--- |
| 發言時限前 30 秒 | 🔔 1 次 |
| 發言時限屆滿 | 🔔🔔 2 次 |
| 緩衝時間（15 秒）屆滿 | 🔔🔔🔔 3 次 |
| 緩衝後再逾時 25 秒 | 🔔🔔🔔🔔🔔 5 次 |

- 自由辯論時，須分別累計兩隊的發言時間，各至 2.5 分鐘為止。
- 若計時出現失誤，立即通知主席，經批准後可讓辯員重新或補足發言時間。

### 發言次序及時間

| 發言順序 | 辯位 | 時間 |
| :--- | :--- | :--- |
| 1 | 正方主辯 | 4 分鐘 |
| 2 | 反方主辯 | 4 分鐘 |
| 3 | 正方一副 | 3 分鐘 |
| 4 | 反方一副 | 3 分鐘 |
| 5 | 正方二副 | 3 分鐘 |
| 6 | 反方二副 | 3 分鐘 |
| 7 | 自由辯論 | 各隊 2.5 分鐘 |
| 8 | 反方結辯 | 4 分鐘 |
| 9 | 正方結辯 | 4 分鐘 |

**自由辯論注意事項**：
- 先由正方發言，雙方交替；每次只可派一位辯員。
- 一方時間用盡後，另一方須派一位辯員用盡其剩餘時間（主席不告知剩餘時間，僅鳴鐘提示）。
- 自由辯論不設緩衝時間。
- 自由辯論結束後，雙方主辯各有 2 分鐘準備作結。

## 三、參賽隊伍

### 賽前準備

- **到場時間**：須於比賽開始前 **10 分鐘** 抵達，逾時未到視作棄權（以香港天文台時間為準）。
- **辯員名單**：須於比賽日期前 **7 日或之前**提交，名單一經提交不接受修改。
  - 名單內辯員不得同時為另一參賽隊伍的辯員。
  - 逾時提交罰分如下：

| 延遲日數 | 扣分 |
| :--- | :--- |
| 1 日 | 2 分 |
| 2 日 | 4 分 |
| 3 日 | 7 分 |
| 4 日或以上 | 8 分 |

### 辯題與否決權

1. 辯題於賽前 **14 日**下午 5 時發放，雙方各有一次否決權（須於翌日下午 4 時前提出）。
2. 第一題被否決後，第二題於賽前 **13 日**發放（未行使否決權的一方可再決定是否使用）。
3. 第二題再被否決，第三題（最終辯題）於賽前 **12 日**發放，不得再否決。
4. 辯題及站方（正反方）均以抽籤決定。

### 比賽規則

- **語言**：必須使用粵語，書名、人名及專有名詞可用外語。違規使用外語，評判將酌量扣分（每次 1–5 分）。
- **行為**：嚴禁人身攻擊、粗言穢語或具冒犯性的行為。
- **台上溝通**：嚴禁與台下觀眾溝通，違者**即時取消參賽資格**。
- **台上物品**：只可攜帶文具、不大於 5×3 英寸的辯卡、或白紙。
- **違禁設備**：嚴禁攜帶具計時、通訊、錄音或攝影功能的設備上台。

### 名單變動（比賽期間）

- 名單以外人士嚴禁上台，違者**取消參賽資格**。
- 如遇突發情況（如辯員缺席），可由名單內其他辯員臨時頂替，但每個更動辯位扣 **5 分**。

### 發言時間
- 主辯（開場）/ 結辯：4 分鐘
- 副辯：3 分鐘
- 自由辯論：每隊 2.5 分鐘（各辯員可自行分配，每次派一位）

發言時間完畢後設 **15 秒緩衝**，緩衝後再逾時 25 秒主席會勒令停止。逾時扣分由第 15.01 秒起計算，每 5 秒扣 3 分，上限 15 分。
